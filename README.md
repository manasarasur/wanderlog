# 🎒 Wanderlog

A travel blog website built on WordPress, showcasing stories, guides, and photography from destinations across India — hill stations, heritage sites, beaches, and food trails.

🔗 **Live Site:** (https://wanderlog-chi.vercel.app/)

## About

Wanderlog is an independent travel publication dedicated to slow travel, cultural immersion, and mindful exploration. Instead of tourist checklists, it focuses on raw, local stories and off-the-beaten-path experiences across India.

## Features

- 🏠 **Home** — Hero section, latest stories, popular destinations, category browsing
- 📝 **Blog** — Full journal listing with categories, search, and sidebar widgets
- 📍 **Destinations** — Explore pages filtered by region (North, South, East, West, Northeast)
- 👤 **About** — Author bio and travel journey
- 🎒 **Travel Tips** — Packing guides, budget tips, safety info, booking advice
- 🖼️ **Gallery** — Masonry photo grid with lightbox
- ✉️ **Contact** — Working contact form with social links

## Tech Stack

- **CMS:** WordPress
- **Theme:** Astra (free)
- **Editor:** Gutenberg (block editor)
- **Plugins:**
  - Contact Form 7 — contact form functionality
  - Yoast SEO — on-page SEO optimization
  - WP Fastest Cache — performance/caching
  - Smush — image compression
  - Envira Gallery Lite — photo gallery lightbox
